# 信号记录（复盘用，港美分文件）

本目录持久化每个交易日发出的信号，**供复盘用**。信号先在对话窗口用表格 + emoji 发出（供用户实时看到、即时操作），**发完之后再把信号内容 append 到本目录的当天文件**——对话流会把信号推走，记录到文件便于事后复盘统计（胜率 / 赔率 / EV 收益）。

## 文件命名（2026-07-15 港美分开）

- 港股信号：`YYYY-MM-DD-HKT-signals.md`
- 美股信号：`YYYY-MM-DD-ET-signals.md`

文件名带 HKT / ET 时区标识，港股、美股信号分别记录，便于按市场分别复盘。

## 规则

- **港美分开、每天每市场一个文件**（如港股 `2026-07-15-HKT-signals.md`、美股 `2026-07-15-ET-signals.md`）。
- **先对话、后记录**：AI 在对话中输出信号（表格 + emoji）+ `alert.sh` 响铃之后，**再把信号内容 append 到对应市场文件**（对话在前、记录在后，时序见 `../SKILL.md`「自主演练模式升级」第 4 条）。
- **每条信号含**：`## 信号类型 · HH:MM:SS · 简述` 标题 + 完整信号内容（**emoji + 字段格式，不画 `=====` 框线、不加粗放大**，与对话窗口一致）+ 可选的结果注释。发信号时间精确到秒。
- **记录字段**：发信号时间、标的、方向、量、参考价、止损、止盈、最大损失、置信度、依据、风险提示（**不写账户 / 资金**，2026-07-15 起 AI 不管账户）。

## 信号类型 emoji（四类）

- 🟢 开仓信号（绿色）
- 🔴 平仓信号（红色）
- 🟡 移动止损（黄色）
- 🟠 移动止盈（橙色）
