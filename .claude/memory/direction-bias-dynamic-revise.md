盯盘方向研判是"起点"不是"全天锚定"——盘中必须随趋势演变动态校准，方向变了就切换策略，绝不固守开盘方向执念。

**Why**：2026-07-17 美股，开盘偏空研判（隔夜暴跌 + 放量，事实无误）后，MU 从 804 连续突破 836/840/853/860 四道阻力（明确转多信号），我却执念"等反弹遇阻做空"2 小时全踏空（每次阻力都被突破、还当成"反弹遇阻"），靠用户提示"为何不做多 + MU 在 VWAP 附近"才修正为做多，错过 804→880 整段。根因：把开盘方向当信仰、缺盘中重估机制 + "反弹"vs"反转"误判（连破多道阻力是反转不是反弹）+ skill「强反弹做空被扫」教训被反向误用（本该转做多却继续等做空）。

**How to apply**：
- **位置预判（主动打头，2026-07-18 用户深化）**：盯盘前不只看大趋势方向，还要看**当前价在大趋势中的位置**——拉 **≥4 个月日线**画高点连线（上沿阻力）+ 低点连线（下沿支撑）通道，当前价接近哪沿即位置判据：下沿超跌区→超跌反弹做多假设（即使大趋势空）、上沿超买区→回调做空假设（即使大趋势多）、中段→顺势。**位置决定盘中方向假设、优先于五维度求和定向**——2026-07-17 MU 开盘 804 处在一个月下跌下沿（超跌区），本该立反弹做多假设，却只看大趋势空就等做空、错过 804→880 深 V。判断大趋势至少 4 个月日线，短周期看不出真通道。
- 反转触发条件（被动兜底，任一即重估方向）：① 连续突破 ≥2 道关键阻力（创新高）= 转多，放弃做空框架、转做多/观望；② 连续跌破 ≥2 道关键支撑 = 转空，放弃做多；③ 标的站上/跌破 VWAP 并企稳 = 日内多空转换。
- 每隔 15-30 分钟、或标的突破关键位时，重跑一次方向研判（趋势/量价/资金流是否仍与开盘方向一致）。
- "反弹"vs"反转"：连破多道阻力 = 反转/趋势性上涨（反弹只是短暂回升），连破 2 道即应重估。
