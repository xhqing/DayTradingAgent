2026-07-19 用户授权：五类信号里的「移动止损」可以简写为「移损」。

**适用场景**：紧凑列举五类信号时用简写——如「开仓 🟢 / 加仓 🔵 / 减仓 🟠 / 平仓 🔴 / 移损 🟡」。完整写法「移动止损」仍然可用、不算错（单独深入讨论移动止损机制时，完整写法更清楚）。

**Why**：「移动止损」四个字在紧凑列举里偏长，用户允许简写为两字「移损」省空间（尤其盯盘实时输出要高密度）。这是用户明确授权的简写，不违反全局「不自创简写」原则——那条针对 AI 擅自造词，用户点名的例外可用。

**How to apply**：列举五类信号时可写「移损」；单独讨论机制时仍用完整「移动止损」表意清楚。
