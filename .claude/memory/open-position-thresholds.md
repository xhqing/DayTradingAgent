开仓信号两道门槛（2026-07-16 用户修订）：

1. **置信度**：只有 🟢 高置信才发开仓信号；🟡 中置信、🔴 低置信**都不发开仓信号**（取代原「中置信→降仓位可发」）。
2. **赔率**：预估赔率 **≥ 1.5** 即可开仓（取代原 ≥2；止盈距离 ≥ 1.5 × 止损距离）。

**Why**：2026-07-16 SOXL 做空（置信度定为 🟡 中、标注了超大单逆势买风险）仍发信号，结果反弹被扫、亏 ~$120（超 max_loss）。用户据此提高置信度门槛——中置信的不发开仓。同时把赔率门槛从 ≥2 降到 ≥1.5：**收紧置信度把关质量、放宽赔率抓更多机会**（用置信度而非赔率做主把关）。

**How to apply**：
- 发开仓信号前两道闸：① 置信度必须 🟢 高（🟡 / 🔴 直接不发，不再降仓位发）② 赔率 ≥ 1.5。两道都过才发 🟢。
- EV 判据相应调整：EV = 胜率×赔率 − 败率×1；赔率 1.5 + 胜率 50% → EV = 0.75 − 0.5 = +0.25（正期望但变薄，更依赖胜率 / 置信度把关，故置信度门槛同步收紧到只发高置信）。
- 置信度 🟢 高的标准（沿用 skill）：信号各维度一致（趋势 + 量价 + 资金流 + 宏观基调同向）、入场点建立在已确认事实（企稳 / 真站稳）、无临近重大事件。
