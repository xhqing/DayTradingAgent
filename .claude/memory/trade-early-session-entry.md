盯盘启动时机直接影响能否抓到日内最佳顺势机会。大涨日（如 2026-07-16 港股恒科 +2.0% 收盘、盘中主升浪 +3.5%）的主升浪集中在早盘 09:30-12:00 完成，午市中段（13:26）才接入时主升浪已走完，之后全天在「高位盘整 / 深回调 / 弱反弹 / 尾盘横盘」里找入场，缺了早盘顺势优势，最终零交易。

**Why**：主升浪是日内胜率 / 赔率最好的顺势机会（开盘资金集中、趋势最顺）；午市后多为回调消化，要么追高（赔率差）、要么接回调飞刀（未企稳）、要么在弱反弹 / 横盘里找赔率不足 2 的机会，难有好入场点。大涨日尾盘空间被早盘消耗，剩余空间常不足以构造赔率 ≥ 2 的交易。

**How to apply**：港股 / 美股盯盘尽量盘前或开盘（港股 09:30 / 美股 21:30）就接入，捕捉主升浪；用户叫醒盯盘时若已过早盘，预期降低、重点等回调企稳而非追高。关联 [[cron-replace-delete-old]]。
